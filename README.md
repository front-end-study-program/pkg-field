# pkg-field

package.json 文件的字段描述项目

| 字段名          | 描述         |
| :-------------- | :----------- |
| name            | 当前包名     |
| version         | 版本号       |
| description     | 描述         |
| main            | 标明入口文件 |
| scripts         | 脚本对象     |
| keywords        | 包搜索关键字 |
| author          | 包作者       |
| license         | 许可证       |
| bugs            | 包相关 issue |
| homepage        | 包首页       |
| dependencies    | 生产依赖包   |
| devDependencies | 线上依赖包   |
